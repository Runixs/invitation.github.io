# 이미지 폴더

이 폴더에 웨딩 사진들을 추가해주세요.

## 필요한 이미지 파일

다음 이름으로 이미지를 추가하세요:

1. **main-photo.jpg** - 메인 대표 사진 (가로형 추천, 최소 1200px 이상)
2. **photo1.jpg** - 갤러리 사진 1 (정사각형 추천)
3. **photo2.jpg** - 갤러리 사진 2 (정사각형 추천)
4. **photo3.jpg** - 갤러리 사진 3 (정사각형 추천)
5. **photo4.jpg** - 갤러리 사진 4 (정사각형 추천)
6. **photo5.jpg** - 갤러리 사진 5 (정사각형 추천)
7. **photo6.jpg** - 갤러리 사진 6 (정사각형 추천)

## 이미지 준비 팁

- **형식**: JPG 또는 PNG
- **크기**:
  - 메인 사진: 1200px x 800px 이상
  - 갤러리 사진: 800px x 800px 이상
- **용량**: 각 이미지 1MB 이하 권장 (웹 로딩 속도를 위해)
- **최적화**: [TinyPNG](https://tinypng.com/) 등으로 압축 추천

## 임시 이미지

실제 이미지를 준비하기 전까지는 다음과 같은 방법으로 임시 이미지를 사용할 수 있습니다:

1. [Unsplash](https://unsplash.com/) 등에서 무료 이미지 다운로드
2. [Placeholder.com](https://placeholder.com/)를 사용하여 임시로 표시
3. 스마트폰으로 촬영한 사진 임시로 사용

위 파일명으로 이미지를 추가하면 자동으로 청첩장에 표시됩니다.
